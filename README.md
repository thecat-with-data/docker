# Docker

A repository of Docker files and setups.

## Initial Project

The initial project is designed to give you the basic gist of setting up a Docker container with an HTML website you can tweak and build.

You may simply navigate to this directory and use `docker compose up --build` with Docker Desktop installed to install and run the program, then navigate to `http://localhost:8080` to verify that your webserver is working.

## Second Project

The second project is designed to give you an understanding of how to set up a Docker container with Python, using Flask.

You may simply navigate to this directory and use `docker compose up --build` with Docker Desktop installed to install and run the program, then navigate to `http://localhost:5000` to verify that your webserver is working.
